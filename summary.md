### What is not a leaderboard submission

This is **not** a leaderboard submission. Caveats:

- Covers one repository (32/500 instances), not the full SWE-bench Verified set.
- Multi-attempt agent protocol, not single-shot evaluation.
- Selection to a single known-manageable repo produces a higher visible
  rate than a random draw across all 500 instances would.

These caveats will appear verbatim in the full bundle README when released.
